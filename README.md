# Grunt

##什么是grunt
JavaScript 世界的构建工具

一句话：自动化。对于需要反复重复的任务，例如压缩（minification）、编译、单元测试、linting等，自动化工具可以减轻你的劳动，简化你的工作。当你在 Gruntfile 文件正确配置好了任务，任务运行器就会自动帮你或你的小组完成大部分无聊的工作。
##安装
~~~sh
// 全局安装
npm install grunt-cli -g
// 项目中安装
npm install grunt --save-dev
// 查看版本
grunt --version
输出
grunt-cli v1.2.0
grunt v1.0.1
~~~
##使用
~~~sh
// 创建gruntfile.js
touch gruntfile.js
~~~
